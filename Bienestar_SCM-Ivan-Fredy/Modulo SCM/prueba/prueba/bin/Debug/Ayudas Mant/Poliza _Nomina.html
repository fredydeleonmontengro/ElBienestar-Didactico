Pasos a seguir

Como podemos observar para el ingreso de datos se deben llenar ciertos campos

Vamos ingresar paso por paso de la siguente manera:

  • 1. Se coloca el No. de Poliza
  • 2. Se coloca la Fecha de Creacion
  • 3. El tipo de Poliza esta automatico
  • 4. Se coloca el No. de Nomina.
  • 5. Se coloca la Fecha de Inicio.
  • 6. Se coloca la Fecga de Finalizacion.
