Pasos a seguir

Como podemos observar para el ingreso de datos se deben llenar ciertos campos

Vamos ingresar paso por paso de la siguente manera:

  • 1. Se coloca el Id del empleado que va a crear la convocatoria.
  • 2. Se coloca el tipo de contratacion que tendrá.
  • 3. Se coloca la fecha que se inicia la convocatoria.
  • 4. Se coloca el medio de comunicacion que se utilizara
