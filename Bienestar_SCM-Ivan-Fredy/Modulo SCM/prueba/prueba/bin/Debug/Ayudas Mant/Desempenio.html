Pasos a seguir

Como podemos observar contamos con una distribución de areas para el desempeño de un empleado dentro de la empresa.
Este formulario cuenta con tres partes importantes: Empleados, Puntaje, Categorías.

  • Indice


    • Seleccionar Empleados

    En este grupo de datos, lo único que se podra seleccionar seria al empleado a evaluar, los demas campos Desempeño y Total se llenarán de forma automatica pero para ello necesita de los demas grupos.
    El boton Insertar guardara en la BD el Desempeño y el Total.


  • Indice


    • ¿Cómo se llená Puntaje?

    En este grupo de Datos se llenara automaticamente al llenar Categorías.


    Nota:
    Las siglas se describiran acontinuación:

    • Puntaje:
      • PT = Puntos Totales
      • PC = Puntos Categorías
      • TC = Total Categorías
    • Desempeño:
      • MDE = Muy debajo de las Expectativas
      • DE = Debajo de las Expectativas
      • AE = Alcanza Expectativas
      • ME = Mejora Expectativas
      • S = Sobresaliente


  • Indice


    • ¿Cómo se llenar Categorías?

    Este grupo de datos de datos se llena de la siguiente manera:
    Las categorías que son: Productividad, Eficiencia, Organizacion, Aprendizaje y Desarrollo, Comunicación, Relaciones Interpersonales y Asistencia. Estas se califican sobre 3 puntos cada una. A excepción de Asistencia que se califica sobre 2 puntos.
    El Desempeño se califica sobre 20 puntos. Este dato de muestra en Puntaje -> TC.
