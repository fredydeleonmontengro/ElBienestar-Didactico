Pasos a seguir

Como podemos observar para el ingreso de datos se deben llenar ciertos campos

Vamos ingresar paso por paso de la siguente manera:

  • 1. Se coloca el Id .
  • 2. Se coloca el Nombre del Medio de Comunicacion.
  • 3. Se coloca ka direccion.
  • 4. Se coloca el Telefono.
  • 5. Se coloca el Estado.

    • Buscar Medio Comunicacion

    Para la busqueda de un medio de comunicacion se utiliza el boton que dice consultar.

    Luego de presionar el boton consultar, aparecera una ventana nueva con un formulario que tendra todos los datos ingresados anteriormente.

    • Funcionamientos extras Cerrar/Ayuda/Minimizar

    Los botones de cerrar, ayuda y eliminar aunque su funcionalidad se utiliza de manera empirica, podemos añadir que estos botones permiten que usted como usuario al momento de estar trabajado en varias areás,pueda minimizar formularios y asi trabajar a la misma vez, dichos son los que se muestran a continuacion.
