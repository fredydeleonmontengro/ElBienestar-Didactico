Pasos a seguir

Como podemos observar contamos con una distribución de areas para modificar las "Las Reuniones".

Vamos ingresar paso por paso de la siguente manera:

  • 1. Consultar el nombre de la reunión.
  • 2. Para luego modificar los datos que sean necesarios.


  • Indice


    • Nombre de la Reunión

    En este apartado se encontrara un campo para ingresar el nombre de la reunión que se desea modificar (luego dar clic al boton "buscar").


  • Indice


    • Busqueda de Reuniones

    En este apartado se mostrara la reunión con los datos registrados para poder modificarlos.


  • Indice


    • Modificación de datos para la Reunión

    En este apartado se mostrarán los datos ya modificados para la reunión, luego se dará clic en el boton Guardar.
